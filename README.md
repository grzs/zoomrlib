## Sequence file format

- header: 48 bytes

### sequencer note syntax:

starts at byte 48

note parameters in 4 bytes:
1. length fraction
2. length whole notes
3. ???
4. channel number (255 means step to the next note)

4 consecutive 255 means end of data

end of file pattern: 06 00 ff 00 00 00 00 EOF
